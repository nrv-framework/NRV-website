## NRV framework scientific citation:
If your code include `import nrv`, you are using the NRV and the developers would be grateful if you would cite the scientific contributions that enforce the framework. The science behind NRV framework and methods used have been published in PLOS Computational Biology. Please cite the framework as:

Couppey T, Regnacq L, Giraud R, Romain O, Bornat Y, et al. (2024) NRV: An open framework for in silico evaluation of peripheral nerve electrical stimulation strategies. PLOS Computational Biology 20(7): e1011826. https://doi.org/10.1371/journal.pcbi.1011826

You can find this list of papers in BibTeX format [here](/assets/citations.bib).

## Citing the code:
The NRV code is recorded in zenodo, you can access it at: https://zenodo.org/records/11520202. You can refer to the code as :
Thomas Couppey, Florian Kolbl, Louis Regnacq, & Roland Giraud. (2024). nrv-framework/NRV: v1.1.0 (v1.1.0). Zenodo. https://doi.org/10.5281/zenodo.11520202
